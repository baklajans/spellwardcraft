# Spellward Craft — калькулятор крафта для ss14 проекта SPELLWARD
 Позволяет редактировать цены базовых ресурсов, считать себестоимость промежуточных и готовых предметов (броня, оружие, инструменты), добавлять их в корзину и получать итог с учётом глобальной наценки и округления.

## Запуск
Использовать онлайн --> https://baklajans.github.io/spellwardcraft/

Для локального запуска достаточно скачать index.html и открыть в браузере, интернет не нужен.

## Возможности
- **Редактируемые цены** базовых ресурсов в карточках.
- **Каталог** с вкладками: Броня, Оружие, Инструменты, Промежуточные ресурсы. Табличный вывод состава и себестоимости, кнопки «В корзину» для готовых предметов.
- **Поиск и фильтры** при добавлении в корзину: карточки с быстрым вводом количества, категории Все/Броня/Оружие/Инструменты.
- **Корзина** с количеством по каждой позиции, кнопками дублирования/удаления и автоматическим пересчётом.
- **Сводки** базовых и промежуточных ресурсов в таблицах.
- **Экспорт прайс-листа** в буфер обмена с учётом наценки и настройки округления для вставки в игру.

## Технологии
- Чистый HTML + CSS + JavaScript
- Tailwind CDN (`@tailwindcss/browser@4`)

## Использование
1. Отредактируйте цены базовых ресурсов в нижних карточках при необходимости.
2. Добавьте предметы через блок «Добавить предмет в корзину» или через таблицы каталога.
3. Настройте наценку и при необходимости снимите/включите округление.
4. Смотрите итоговую стоимость, а также таблицы требуемых базовых и промежуточных ресурсов.
5. Нажмите «Экспорт» в шапке, чтобы скопировать прайс-лист готовых предметов с учётом наценки и округления.
